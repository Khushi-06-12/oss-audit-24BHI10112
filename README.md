## Scripts Description

### Script 1 - System Identity Report
Displays system information like user, kernel, uptime, and date.

### Script 2 - FOSS Package Inspector
Checks if a package is installed and shows its details.

### Script 3 - Disk and Permission Auditor
Shows permissions and size of important directories.

### Script 4 - Log File Analyzer
Counts number of "error" lines in a log file.

### Script 5 - Open Source Manifesto Generator
Takes user input and generates a manifesto text file.

---

## How to Run Scripts

1. Open terminal in Linux
2. Give permission:
   chmod +x script.sh

3. Run script:
   ./script.sh
